# Cross-method comparisons — verification scripts for paper findings

Scripts that recompute the headline numbers cited in the "What embeddings
recover" subsection of the paper from the per-method distance matrices stored
under `analysis/<method>/experiments/<experiment>/evaluation_results/flores/`.

The scripts read the saved `distances.csv` files and print findings to stdout;
they do not retrain anything. Re-running them is fast (seconds).

## "Trustworthy" methods

Following the paper's "Kept" column in Table 2 (silhouette over four
typological macro-families), the analyses below restrict the cross-method
counts to the methods whose representation space recovers the four
macro-families at all. Two cardinalities are used:

- **10 methods** — the 8 surface methods (TF-IDF char/word × normalized/native,
  FastText normalized/native, Word2Vec normalized/native) plus the two
  fine-tuned encoders that retain macro-family structure (`CANINE-MLM`,
  `XLM-R MLM`). Used by `01_trustworthy_consensus.py`, which checks
  qualitative paper claims.
- **9 methods** — same as above without `XLM-R MLM`, which has a
  near-zero (negative) macro-family silhouette and a Mantel-insignificant
  LDND correlation. Used by `02-04`, where finer per-pair comparisons
  require methods that pass both checks.

Per-experiment silhouette scores live at
`analysis/<m>/experiments/<exp>/evaluation_results/flores/.../silhouette_report.txt`;
Mantel p-values for the LDND correlation are in
`gold/_correlations/correlation_lexicostat_ldnd_with_bootstrap.csv`.

## Scripts

| Script | Question it answers |
|---|---|
| `scripts/01_trustworthy_consensus.py` | For the four headline claims (Lombard/Venetian central, Sardinian-Iberian top-3, Slovenian/Croatian nearest dialect = Venetian), how many of the 10 trustworthy methods agree, and which diverge? |
| `scripts/02_bidirectional_gold.py` | For each of the 18 (dialect, historical contact) pairs in the qualitative reference matrix, how many of the 9 trustworthy methods place the contact in the dialect's top-3 nearest external (Dir1) and/or place the dialect in the contact's top-3 nearest dialect (Dir2)? |
| `scripts/03_per_method_recovery.py` | Per-method tally of how many of the 18 pairs each method recovers in either direction. |
| `scripts/04_comprehensive_analysis.py` | Full breakdown: top-1 external per dialect, position of every curated contact, top-1 dialect per external standard, dialect integration ranking. |

Run from the repo root, e.g.:

```bash
python -m analysis._comparisons.scripts.01_trustworthy_consensus
python -m analysis._comparisons.scripts.02_bidirectional_gold
python -m analysis._comparisons.scripts.03_per_method_recovery
python -m analysis._comparisons.scripts.04_comprehensive_analysis
```

## Reference matrix used by 02-04

The qualitative historical-influence matrix is hardcoded inside the scripts
(it is also reproduced as Appendix C of the paper):

| Dialect | Historical-contact standards |
|---|---|
| Friulian (fur) | deu, slv, hrv |
| Ligurian (lij) | oci, spa, fra |
| Lombard (lmo) | deu, oci, spa |
| Sardinian (sc) | spa, cat, fra |
| Sicilian (scn) | spa, fra, cat |
| Venetian (vec) | hrv, slv, deu |

## outputs/

Auxiliary breakdowns produced during paper drafting from the same distance
matrices: per-method top-1 dialect view, per-family attraction scores,
integration ranks, and silhouette scores under three alternative partitionings.
Not regenerated by the scripts above, but recomputable from the same
`distances.csv` files.
